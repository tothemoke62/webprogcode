Miről szól az alkalmazás?

A Net Pizzéria éjjel-nappal nyitva tart. Rendeléseket SMS-ben fogadunk, egyféle pizza rendelhető egy üzenetben. Az alkalmazás bemutatja a pizzák, kategóriák és rendelések kezelését.

🍕

Pizzák

Többféle kategória, vegetáriánus lehetőségekkel.

📋

Kategóriák

Minden kategóriának saját ára van.

🛵

Rendelések

Rendelés felvétele és kiszállítás nyomon követése.

Az alkalmazás részei

JavaScript CRUD

Pizzák kezelése tömbből, tiszta JavaScript-tel.

React CRUD

React komponensek, useState állapotkezeléssel.

SPA

Egyoldalas alkalmazás, két React mini-applikációval.

Fetch API CRUD

JavaScript + PHP backend, MySQL adatbázissal.

Axios CRUD

React + Axios, szerver oldali adatkezeléssel.

OO JavaScript

Objektumorientált grafikus JavaScript alkalmazás.
